# Syphinium
Jogo plataforma 2D desenvolvido no unity, onde você controla um personagem que deve explorar o mundo de Syph, conta com diversos mapas e inimigos, sistema de save, inventario, diálogos, etc...

# Como Jogar?

Faça o <a href="https://drive.google.com/file/d/1CUH1Kszb2ew0aAv7v3kjdLVcT2988B49/view?usp=sharing">Download</a> do ZIP "Syphinium.rar", Extraia o mesmo e execute o arquivo "Syphinium.exe".
