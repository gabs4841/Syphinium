# Syphinium
Jogo plataforma 2D desenvolvido no unity, onde você controla um personagem que deve explorar o mundo de Syph, conta com diversos mapas e inimigos, sistema de save, inventario, diálogos, etc...

# Como Jogar?

Faça o <a href="https://drive.google.com/file/d/1c_3Gpgcx5sXAnyy3UvJGfRn72Xao6gS_/view?usp=sharing">Download</a> do "Syphinium.zip", Extraia o mesmo e execute o arquivo ![image](https://github.com/gabs4841/Syphinium/assets/74026100/4eb40f55-8d5a-4d1a-90f1-ea94c68aa21a) "Syphinium.exe".
